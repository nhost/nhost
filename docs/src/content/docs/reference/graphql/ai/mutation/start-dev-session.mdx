title: startDevSession

```graphql
startDevSession: Session!
```

Create a developer session

## Permissions needed

Only admins can create developer sessions



## Response

Type: `Session!`





    
    - **`sessionID`** (`String!`): ID of the session
    
    - **`assistantID`** (`String!`): ID of the assistant used in this session
    
    - **`userID`** (`uuid!`): ID of the user who started this session
    
    - **`createdAt`** (`timestampz!`): Messages in this session
    



**Request Example:**

```graphql query
query startDevSession {
  graphite {
    startDevSession {
      sessionID
      assistantID
      userID
      createdAt
    }
  }
}
```
