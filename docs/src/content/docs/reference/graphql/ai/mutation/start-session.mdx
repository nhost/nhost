---
title: startSession
---

```graphql
startSession {
   assistantID: String! 
}: Session!
```

Create a session with a given assistant

## Permissions needed

select (sessions):
- id
insert (sessions):
- user_id
- assistant_id


## Arguments



    - **`assistantID`** (`String!`): Start a session with this assistant



## Response

Type: `Session!`





    
    - **`sessionID`** (`String!`): ID of the session
    
    - **`assistantID`** (`String!`): ID of the assistant used in this session
    
    - **`userID`** (`uuid!`): ID of the user who started this session
    
    - **`createdAt`** (`timestampz!`): Messages in this session
    



**Request Example:**

```graphql query
query startSession (
  $assistantID: String!
) {
  graphite {
    startSession (
      assistantID: $assistantID
    ) {
      sessionID
      assistantID
      userID
      createdAt
    }
  }
}
```
