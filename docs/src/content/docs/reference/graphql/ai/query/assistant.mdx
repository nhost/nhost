title: assistant

```graphql
assistant {
   assistantID: String! 
}: Assistant
```

Retrieve an assistant

## Permissions needed

select (assistants):
- id
- assistantID


## Arguments



    - **`assistantID`** (`String!`): ID of the assistant to retrieve



## Response

Type: `Assistant`





    
    - **`assistantID`** (`String!`): ID of the assistant
    
    - **`name`** (`String!`): Name of the assistant
    
    - **`description`** (`String!`): Description of the assistant
    
    - **`instructions`** (`String!`): Instructions for the assistant. This is used to instruct the AI assistant
on how to behave and respond to the user

**Request Example:**

```graphql query
query assistant (
  $assistantID: String!
) {
  graphite {
    assistant (
      assistantID: $assistantID
    ) {
      assistantID
      name
      description
      instructions
      model
      graphql {
        name
        description
        query
        arguments {
          name
          description
          type
          required
          }
        }
      webhooks {
        name
        description
        URL
        arguments {
          name
          description
          type
          required
          }
        }
    }
  }
}
```
