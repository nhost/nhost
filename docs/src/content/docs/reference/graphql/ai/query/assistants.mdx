title: assistants
---

```graphql
assistants: [Assistant!]!
```

Retrieve all assistants

## Permissions needed

select (assistants):
- id
- assistant_id



## Response

Type: `[Assistant!]!`





    
    - **`assistantID`** (`String!`): ID of the assistant
    
    - **`name`** (`String!`): Name of the assistant
    
    - **`description`** (`String!`): Description of the assistant
    
    - **`instructions`** (`String!`): Instructions for the assistant. This is used to instruct the AI assistant
on how to behave and respond to the user
    
    - **`model`** (`String!`): Model to use for the assistant.

        }

          }

  }
