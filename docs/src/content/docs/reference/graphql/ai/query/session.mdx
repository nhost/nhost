---
title: session
---

```graphql
session {
   sessionID: String! 
}: Session
```

Retrieve a session

## Permissions needed

select (sessions):
- id
- assistant_id
- session_id
- user_id


## Arguments



    - **`sessionID`** (`String!`): ID of the session to retrieve



## Response

Type: `Session`





    
    - **`sessionID`** (`String!`): ID of the session
    
    - **`assistantID`** (`String!`): ID of the assistant used in this session
    
    - **`userID`** (`uuid!`): ID of the user who started this session
    
    - **`createdAt`** (`timestampz!`): Messages in this session
    



**Request Example:**

```graphql query
query session (
  $sessionID: String!
) {
  graphite {
    session (
      sessionID: $sessionID
    ) {
      sessionID
      assistantID
      userID
      createdAt
    }
  }
}
```
