The **Metrics** tab shows performance data for an individual function — how often it is invoked, how fast it responds, how much data it returns, and how often it fails — over a time range you choose.

:::note
This is a Pro/Team/Enterprise feature. This is not available on Starter projects.
:::

:::tip
This tab is a focused view for a single function. For project-wide observability across all your services, see [Metrics with managed Grafana](/platform/cloud/metrics).
:::

## Viewing Metrics

Metrics require at least one deployed function that has received traffic. If you don't have one yet, follow [Getting Started](/products/functions/guides/getting-started) to create and deploy your first function, or [Generating Metrics](/products/functions/guides/generating-metrics) for a sample function that populates every chart.

Open the <a href="https://app.nhost.io" target="_blank" rel="noopener noreferrer">Nhost dashboard</a> and go to **Functions** > *your function* > **Metrics**. Metrics are scoped to the function you are viewing.

Use the time range filter at the top to choose the window, and the refresh button to reload the latest data.

## Summary

Three cards summarize the selected time range:

| Card | Description |
| ---- | ----------- |
| **Total Number of Invocations** | How many times the function ran |
| **Total Bytes Sent** | Combined size of all responses |
| **Total Duration** | Combined execution time across all invocations |

## Charts

Charts are grouped into three sections. Series are broken down by HTTP method, except where noted.

### General

- **Invocations** — number of invocations, by HTTP method.
- **Response Status** — number of invocations, by response status code.
- **Average Response Size** — average bytes returned per response.

### Response Times

- **Maximum Response Time** — the slowest response in each interval.
- **Response times 95th percentile** — 95% of responses complete within this time.
- **Response times 75th percentile** — 75% of responses complete within this time.
- **Average Response Time** — mean response time.

### Errors

- **Error Rate** — failed invocations as a share of total invocations.

## Time Range

Pick a preset or an absolute range from the filter at the top:

- **Presets** — Last 5 minutes, 15 minutes, 30 minutes, 1 hour, 3 hours, 6 hours, 12 hours, 24 hours, and 7 days. The default is the last 6 hours.
- **Absolute** — set explicit **From** and **To** timestamps.

You can also zoom directly on any chart:

- **Drag** horizontally across a chart to zoom into that window.
- **Double-click** a chart to zoom out — the visible span doubles, centered on the current window, up to a maximum of 90 days.

## Interacting with Charts

- **Pin the tooltip** — click a data point to pin the hover tooltip in place so you can read exact values; click again or press `Esc` to unpin.
- **Toggle series** — click a legend item to isolate that series; click it again to restore all series. Hold `⌘`/`Ctrl`/`Shift` while clicking to show or hide individual series.
- **Expand a chart** — open any chart into a full-width panel for a closer look.
- **Share a view** — the selected time range, expanded panel, and hidden series are stored in the page URL, so you can copy the link to share the exact view with a teammate.
