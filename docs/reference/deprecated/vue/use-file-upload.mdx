Use the composable `useFileUpload` to upload a file.

```ts
const {
  add,
  upload,
  cancel,
  isUploaded,
  isUploading,
  isError,
  progress,
  id,
  bucketId,
  name
} = useFileUpload()

const handleFormSubmit = async (e) => {
  e.preventDefault()

  await upload({ file })
}
```
