---
title: startSession
---

```graphql
startSession {
   assistantID: String! 
}: Session!
```

Create a session with a given assistant

## Permissions needed

select (sessions):
- id
insert (sessions):
- user_id
- assistant_id


## Arguments



    <ParamField body="assistantID" type="String!" required>
    Start a session with this assistant
    </ParamField>



## Response

Type: `Session!`





    
    <ResponseField name="sessionID" type="String!">
    ID of the session
    </ResponseField>
    
    <ResponseField name="assistantID" type="String!">
    ID of the assistant used in this session
    </ResponseField>
    
    <ResponseField name="userID" type="uuid!">
    ID of the user who started this session
    </ResponseField>
    
    <ResponseField name="createdAt" type="timestampz!">
    Messages in this session
    </ResponseField>
