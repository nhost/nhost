```graphql
session {
   sessionID: String! 
}: Session
```

Retrieve a session

## Permissions needed

select (sessions):
- id
- assistant_id
- session_id
- user_id


## Arguments



    <ParamField body="sessionID" type="String!" required>
    ID of the session to retrieve
    </ParamField>



## Response

Type: `Session`





    
    <ResponseField name="sessionID" type="String!">
    ID of the session
    </ResponseField>
    
    <ResponseField name="assistantID" type="String!">
    ID of the assistant used in this session
    </ResponseField>
    
    <ResponseField name="userID" type="uuid!">
    ID of the user who started this session
    </ResponseField>
    
    <ResponseField name="createdAt" type="timestampz!">
    Messages in this session
    </ResponseField>
