We are excited to unveil our newly revamped [React Apollo example project](https://react-apollo.example.nhost.io) along
with a powerful new template designed to accelerate your project development. This release marks a significant step in
our commitment to providing valuable tools and resources for developers working with React and Nhost.

## 🌟 New React Example Porject

Our example project has been rewritten from scratch using the popular UI components library
[shadcn/ui](https://ui.shadcn.com/).

### 📋 Features

The examples showcases the following featureset:

1. 🔒 Authentication and protected pages via:

   - Email and password
   - Social login (Google, Apple, Github...)
   - WebAuthn
   - Magic link
   - Anonymous login

## 🏗️ New Nhost React Apollo Template

To make it even simpler and faster to get started working on your next project with Nhost and React, we've published
a new [React Apollo Template](https://www.npmjs.com/package/@nhost/cra-template-react-apollo) incorporating features
from the example project.

Getting started with the template is easy:

1. Scaffold the template

```shell
npx create-react-app my-app --template @nhost/react-apollo
```

2. Replace the '\<subdomain\>' and '\<region\>' in `src/index.tsx` with you project's details

3. Start the project

```shell
npm start
```
