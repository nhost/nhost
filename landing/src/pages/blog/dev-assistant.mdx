In the world of software development, tools that simplify and assist the development process are increasingly valuable. Developer assistants can help us go from idea to production faster, and with less errors. 

Nhost’s new **Dev Assistant** is integrated into the Dashboard and it is designed to support you with both SQL and GraphQL.

### Database and GraphQL API

The Nhost Dev Assistant understands both the Postgres database schema and the GraphQL API schema. This allows it to provide tailored, context-aware suggestions and solutions.

#### Example Interactions

User: "How do I create a new user in my database?"

Assistant: "To create a new user, you can use this SQL command: `INSERT INTO users (name, email) VALUES ('Alice', 'alice@example.com');`. This command inserts a new user into your 'users' table."

User: "Can you show me how to fetch user details using GraphQL?"

Assistant: "Sure! You can use the following GraphQL query: `{ user(id: 1) { name, email } }`. This query retrieves the name and email of the user with an ID of 1."
