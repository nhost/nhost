## Introduction

React Flow is a small company based in Berlin that focuses on developing a frontend library for building node-based interfaces, workflow editors, and interactive diagrams. The library is widely used, and to secure its financial stability, the company has introduced a subscription payment model powered by Nhost.

## Problem

The team at React Flow consists of frontend developers with limited backend knowledge. Therefore, they needed a solution to handle complicated auth functionality with ease.

## Solution

The team chose Nhost over its competitors as it integrated well with React Flow's stack and offered them all necessary features through its SDKs and GraphQL API. The active support team on Discord with fast response times was crucial in ensuring React Flow's confidence in their choice.

## Benefits of Nhost

Using Nhost enabled React Flow to build its subscription platform in just two months with one frontend developer working on it. The team made heavy use of the NextJS integration, and access to the Hasura console was helpful for debugging. The ability to test features and make changes in the dev environment before deploying with a simple Git command, was also important to ensure a good release process for the team.

## Conclusion

React Flow recommends Nhost to other developers and teams, especially frontend (React) developers who need reliable authentication and database for any project size. Paired with great SDKs for React and Next.js.
