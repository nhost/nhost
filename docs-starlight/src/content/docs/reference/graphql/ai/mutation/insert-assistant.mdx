---
title: insertAssistant
---

```graphql
insertAssistant {
   object: AssistantInput! 
}: Assistant!
```

Create an assistant

## Permissions needed

select (assistants):
- id
insert (assistants):
- user_id


## Arguments



    - **`object`** (`AssistantInput!`): Assistant to create
