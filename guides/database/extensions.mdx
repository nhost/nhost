## Featured Extensions

Below are detailed descriptions of some of our most popular and powerful extensions.

### PostGIS

<Note>
PostGIS is the leading spatial database extender for PostgreSQL, adding support for geographic objects and location queries.
</Note>

#### Features
- Store and query geographic data
- Perform spatial calculations
- Support for various coordinate systems
- Advanced spatial indexing

#### Managing PostGIS

To install the extension:

```sql SQL
SET ROLE postgres;
CREATE EXTENSION postgis;
```

To uninstall:

```sql SQL
SET ROLE postgres;
DROP EXTENSION postgis;
```

**Resources**
- [Official website](https://postgis.net/)

### pgvector

<Note>
pgvector enables vector similarity search in Postgres, making it ideal for AI and machine learning applications.
</Note>

#### Key Capabilities
- Exact and approximate nearest neighbor search
- L2 distance, inner product, and cosine distance support
- Compatible with any Postgres client
- ACID compliance and point-in-time recovery

#### Managing pgvector

To install:

```sql SQL
SET ROLE postgres;
CREATE EXTENSION vector;
```

To uninstall:

```sql SQL
SET ROLE postgres;
DROP EXTENSION vector;
```

**Resources**
- [GitHub Repository](https://github.com/pgvector/pgvector)
